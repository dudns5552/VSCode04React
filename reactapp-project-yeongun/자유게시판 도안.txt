자유게시판 구현목록


<리스트>
1. 제목 클릭하면 해당 글로 이동( view전환)
2. 글쓰기 버튼
3. 페이지 목록버튼
4. 게시물 정렬(인덱스나 게시물 시간별 정렬)
5. 게시물 날짜에 따라 날짜 또는 시간으로 표현(오늘이면 시간, 아니면 날짜)

<읽기모드>
1. 목록버튼(lists전환)
2. 이전글(인덱스 줄여서 이동버튼)
3. 다음글(인덱스 더해서 이동버튼)
4. 자기글일 경우 수정, 삭제 버튼(파이어베이스의 작성자 id와 로그인 아이디 비교해서 토글해야될듯)
5. 수정버튼 클릭시 수정모드로 전환
6. 삭제버튼 클릭시 물어보고 삭제 진행


<쓰기모드>
제목과 내용만 작성가능 작성자, Date는 히든으로 값 입력후
파이어베이스에 등록
인덱스는 전송시점의 파이어스토어의 마지막게시물의 인덱스를 얻어온후
+1해서 등록



<수정모드>
쓰기와 폼은 같고 파이어베이스에서
